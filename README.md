# web-books-server
Web Books Server — самоорганизующийся сервер для личной коллекции электронных книг. Работает с INPX-каталогами, парсит их в SQLite, предоставляет удобный веб-интерфейс для поиска, группировки по авторам/сериям и скачивания. Поддерживает OPDS, избранное, многопользовательский режим, темную тему и горячие клавиши.
